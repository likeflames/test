💻 代码高亮

自动语法高亮，支持行号：


function fibonacci(n) {
  if (n <= 1) return n;          // 第 2 行高亮
  return fib(n-1) + fib(n-2);    // 第 3 行高亮
}

data-line-numbers="3,8-10" 指定高亮行

💻 逐步高亮


def quicksort(arr):              # 第 1 步
    if len(arr) <= 1:            # 第 2 步
        return arr
    pivot = arr[0]               # 第 3 步
    left = [x for x in arr[1:] if x < pivot]
    right = [x for x in arr[1:] if x >= pivot]
    return quicksort(left) + [pivot] + quicksort(right)

切换步骤时高亮行同步更新

▶️ 自动播放模式

适合展厅、信息亭、无人值守展示

Reveal.initialize({
  autoSlide: 5000,     // 每 5 秒翻页
  autoSlideStoppable: false,
  loop: true           // 循环
});

每页可单独设置时间：

<section data-autoslide="10000">

按 A 暂停/继续
