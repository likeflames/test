📦 第一章 项目初始化

使用 VitePress + Teek 主题创建博客项目

1. 创建项目
npm create vitepress@latest my-site
cd my-site
2. 安装 Teek 主题
pnpm add vitepress-theme-teek
3. 目录结构
my-site/
├─ docs/
│  ├─ .vitepress/          # 配置文件
│  │  ├─ config.ts         # VitePress 配置
│  │  ├─ teekConfig.ts     # Teek 主题配置
│  │  └─ theme/            # 主题自定义
│  ├─ @pages/              # 功能页面
│  ├─ public/              # 静态资源
│  └─ index.md             # 首页

⚙️ teekConfig.ts 基础配置

import { defineTeekConfig } from "vitepress-theme-teek/config";

export const teekConfig = defineTeekConfig({
  teekHome: true,           // 启用博客首页
  banner: {
    enabled: true,
    name: "likeflames",
    bgStyle: "fullImg",     // 全屏背景图
    imgSrc: ["/bg/xxx.png"],// 轮播图片
    description: ["欢迎来到我的网站"],
  },
  author: { name: "likeflames" },
  footerInfo: {
    copyright: { createYear: 2026, suffix: "likeflames" },
  },
});

teekConfig.template.ts 是配置参考手册，不要删除

🔧 本地开发命令

启动开发服务器
pnpm docs:dev

重要 每次修改代码后先 pnpm docs:build 再推送

🚀 第三章 Vercel 部署

1. 推送代码到 GitHub
git init
git add .
git commit -m "first commit"
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
2. Vercel 导入项目

打开 https://vercel.com → Add New → Project

导入 GitHub 仓库 → Framework 选 VitePress → Deploy

3. 自动部署

每次 git push 后 Vercel 自动构建部署

构建命令：TZ=Asia/Shanghai pnpm docs:build

🌐 域名绑定

阿里云 DNS 设置

域名解析添加以下记录：

记录类型: A          主机记录: @     记录值: 76.76.21.21
记录类型: CNAME     主机记录: www   记录值: cname.vercel-dns.com
Vercel 绑定域名

项目 Settings → Domains → 输入 likeflames.online

Vercel 会自动验证并配置 SSL 证书

提示 等待 1-10 分钟 DNS 生效

📝 第四章 文章发布

Frontmatter 格式
---
title: 文章标题
date: "2026-05-07"        // 日期必须加引号
categories: [分类名]
tags: [标签1, 标签2]
author: likeflames
description: 文章简介（用于 SEO）
permalink: /english-slug   // 固定链接（推荐）
---
⚠️ 日期时区注意事项

date 必须加引号（如 "2026-05-07"）

否则 YAML 会解析为 UTC 日期，导致显示少一天

同时 teekConfig.ts 中设置 dateUTC: false

🗄️ MongoDB Atlas 注册

步骤

1. 打开 https://mongodb.com → 注册账号

2. 创建集群 → 选择 M0 Sandbox（免费）

3. 区域选 Singapore（靠近中国）

4. Database Access → 创建数据库用户，记下密码

5. Network Access → 添加 0.0.0.0/0（允许所有 IP）

6. Connect → 复制连接字符串

mongodb+srv://用户名:密码@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority

☁️ Twikoo 后端部署

一键部署到 Vercel

打开这个链接（用 GitHub 账号登录）：

https://vercel.com/new/clone?repository-url=https://github.com/twikoojs/twikoo/tree/main
Vercel 项目设置

Settings → General → Root Directory 设为 src/server/vercel-min

添加环境变量：MONGODB_URI = 刚才复制的连接字符串

Settings → Deployment Protection → Vercel Authentication 设为 Disabled

Deployments → Redeploy

成功标志 访问部署 URL 显示 "Twikoo 云函数运行正常"

🔗 配置 DNS 子域名

为 Twikoo 后端配置独立域名，加速国内访问：

阿里云 DNS

CNAME 记录
twikoo → cname.vercel-dns.com

Vercel 绑定

项目 Settings → Domains
添加 twikoo.likeflames.online

最终 envId = https://twikoo.likeflames.online

🧩 前端集成 Twikoo

1. 安装依赖
pnpm install twikoo
2. 创建组件
// docs/.vitepress/theme/components/Twikoo.vue
<script setup>
import { onMounted, watch } from 'vue'
import { useRoute } from 'vitepress'
const route = useRoute()
const initTwikoo = async () => {
  if (typeof window !== 'undefined') {
    const twikoo = await import('twikoo')
    twikoo.init({ envId: 'https://twikoo.likeflames.online', el: '#twikoo' })
  }
}
watch(route, () => setTimeout(initTwikoo, 500))
onMounted(() => initTwikoo())
</script>
<template><div id="twikoo"></div></template>

🧩 Layout.vue 注册

// docs/.vitepress/theme/components/Layout.vue

import Teek from "vitepress-theme-teek"

📱 归档页贡献图适配

核心改动：移动端只显示 3 个月
// ContributeChart.vue 中动态计算范围
const now = new Date();
const year = now.getFullYear();
const pad2 = (n) => String(n).padStart(2, "0");
const monthStart = new Date(year, now.getMonth() - 2, 1);
const monthEnd = new Date(year, now.getMonth() + 1, 0);
const mobileRange = [
  `${year}-${pad2(monthStart.getMonth()+1)}-${pad2(monthStart.getDate())}`,
  `${year}-${pad2(monthEnd.getMonth()+1)}-${pad2(monthEnd.getDate())}`,
];

桌面端：显示 12 个月数据

移动端：显示最近 3 个月（完整月份），优化适配

📧 邮箱通知配置

SMTP 设置（163 邮箱）

1. 登录 163 邮箱 → 设置 → POP3/SMTP/IMAP

2. 开启 SMTP 服务 → 获取授权码

3. 管理面板填入：

SENDER_EMAIL: iamwujianhua@163.com
SMTP_SERVICE: 163
SMTP_USER:    iamwujianhua@163.com
SMTP_PASS:    授权码（不是登录密码）

验证 管理面板底部的 "Email notification test" 可发测试邮件

😊 表情包配置

JSON 格式
{
  "😃": {                    // 标签名（支持 Emoji）
    "type": "emoji",
    "container": [
      { "icon": "😂", "text": "" }
    ]
  },
  "<img src='图片URL' style='...'>": {  // 图片标签
    "type": "image",
    "container": [
      { "icon": "<img src='表情图'>", "text": "表情名" }
    ]
  }
}

多套表情包可合并到一个 /emotions.json 文件中

管理面板 EMOTION_CDN 填 /emotions.json

🔐 管理员密码与安全

首次配置

点评论区 ⚙️ → 设置管理员密码

⚠️ 第一次设密码前，任何人都可以成为管理员！

忘记密码

Vercel 环境变量加 RESET_PASSWORD=true → Redeploy

设完新密码后删除该变量 → 再 Redeploy 一次

个人信息配置

管理面板设置 BLOGGER_EMAIL + BLOGGER_NICK

评论时邮箱匹配自动显示「站长」标签
