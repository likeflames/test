📊 如何制作 HTML PPT

从创建文件到推送上线，完整流程

← → 翻页 · ⛶ 全屏

1

复制模板

在 docs/public/ppt/ 目录下复制 template.html

cp docs/public/ppt/template.html docs/public/ppt/我的演示.html

或者直接用 VS Code 打开模板另存为新文件

← 上页 · 下页 →

2

修改内容

找到 HTML 中 <section> 标签，每个 <section> 就是一页幻灯片：

<section id="s1" class="page">
  <h1>标题</h1>
  <p>内容</p>
</section>

<section id="s2" class="page">
  <h2>第二页</h2>
  <p>内容</p>
</section>

← 上页 · 下页 →

3

添加导航链接

每页底部要添加上一页/下一页的链接，使用锚点跳转：

<a href="#s1">← 上页</a>
<a href="#s3">下页 →</a>

注意 每页的 id 要唯一，不能重复

← 上页 · 下页 →

4

本地预览

在浏览器中直接双击打开 HTML 文件就能预览效果

确认所有页面显示正常、链接跳转正确

或者用 VS Code 的 Live Server 插件实时预览

← 上页 · 下页 →

5

添加导航入口

在 docs/@pages/ppt.md 中添加新 PPT 的卡片：

<div class="ppt-card"
  onclick="location.href='/ppt/我的演示.html'">
  <div class="ppt-card-icon">📄</div>
  <div class="ppt-card-body">
    <h3>我的演示</h3>
    <p>描述文字</p>
  </div>
</div>

← 上页 · 下页 →

6

推送上线

在终端执行以下命令：

# 添加文件
git add docs/public/ppt/我的演示.html
git add docs/@pages/ppt.md

# 提交并推送
git commit -m "feat: add new ppt"
git push

等待 Vercel 自动部署（约 1-2 分钟）

← 上页 · 下页 →

7

访问展示

部署完成后，通过以下方式访问：

  • 网站导航栏 → PPT → 点击对应卡片
  • 或直接输入网址：
    https://likeflames.online/ppt/我的演示.html

播放时按 F11 全屏，← → 翻页
